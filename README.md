Cartify – Shopping Cart Web Application

Cartify is a contemporary e-commerce shopping cart web application developed to enhance the shopping experience. Users can shop products, add or remove items in the cart, change product quantities, and efficiently manage their shopping carts in real time.

The Cartify app is created using React.js in the front end and uses Redux Toolkit for effective state management. The app is highly scalable, fast, and has an excellent user interface.

✨ Key Features
Add/remove items from the shopping cart
Modify product quantity
Calculate cart totals in real time
Manage states using Redux Toolkit
Efficient user interface
Toast notifications
🛠️ Tech Stack
React.js
Redux Toolkit
JavaScript (ES6+)
Tailwind CSS/CSS
React Toastify










# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
